CPF Utils

O seu utilitário para manipulação de CPF.

Biblioteca JavaScript completa para gerar, validar e formatar CPFs com funcionalidades avançadas.

Instalação da biblioteca

  • NPM
  • Bun
  • PNPM
  • Yarn
npm install cpf-utils check
bun add cpf-utils check
pnpm add cpf-utils check
yarn add cpf-utils check
